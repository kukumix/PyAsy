PyAsy
=====

PyAsy is a Python wrapper of the Asymptote_ vector graphics language.

Contributing
------------

Contributions are welcome!  Please send comments, suggestions, and/or
patches to the primary author (`Matthew Emmett`_).  You will be credited.

If you plan to extend or modify PyAsy in a more substantial way,
please see the `PyAsy project page`_.
